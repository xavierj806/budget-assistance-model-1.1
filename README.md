# budget-assistance-model-1.1
The model helps in planning a trip based on peoples preference and budget in .goa, India.
Overview

This repository contains an end-to-end machine learning project implemented in Python using Jupyter Notebook.
The project demonstrates the complete machine learning workflow, including data preprocessing, feature engineering, model training, evaluation, and model persistence.

It is designed as a portfolio project to showcase practical machine learning and data science skills.
